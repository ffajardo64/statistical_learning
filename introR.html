Introdução à ciência de dados

O curso tem como objetivo discutir diferentes metodologias para estudar e analisar conjuntos de dados usando a linguagem R. O curso é dividido em várias aulas teórico-práticas, sendo que cada aula é composta por scripts com exemplos práticos para acompanhar os conceitos tratados nas aulas.

Aula 0

Março, 2023

Plano de Ensino

Plano de ensino proposto para a disciplina. Nesse material é informado o programa da disciplina e as referências bibliográficas sugeridas para o acompanhamento dos assuntos.

Aula 1

Março, 2023

Introdução I: Estruturas de dados e operadores de atribuição

Nessa aula é apresentada uma introdução à linguagem R. São discutidos os principais conceitos da linguagem, como estruturas de dados, operadores de atribuição e alguns exemplos são discutidos.

Aula 2

Março, 2023

Estruturas de controle de fluxo

Nessa aula são discutidos os principais conceitos de estruturas de controle de fluxo, como laços de repetição e condicionais.

Aula 3

Abril, 2023

Funções.

Nessa aula são discutidos os conceitos de funções em R, como criar funções, passagem de parâmetros, retorno de valores, funções anônimas e infixas. Assim como são discutidos os conceitos de escopo de variáveis e de passagem de parâmetros por valor e por referência.

Aula 4

Abril, 2023

Simulação Estocástica I.

Nessa aula é apresentado o conceito de simulação estocástica. São discutidos alguns algoritmos para geração de valores pseudo-aleatórios, ainda são apresentados alguns geradores de números pseudo-aleatórios uniformes.

Aula 5

Abril, 2023

Simulação Estocástica II

Nessa aula são discutidos os geradores congruenciais e a sua utilidade na geração de valores pseudo-aleatórios. São discutidos alguns exemplos práticos do uso da simulação de Monte Carlo.

Aula 6

Abril, 2023

Simulação Estocástica III.

São apresentadas algumas técnicas para geração de números pseudo-aleatórios não-uniformes e vários exemplos práticos são apresentados para o acompanhamentos dos conceitos.
